Astro is a new kind of static site generator that delivers lightning-fast performance by default. It uses a unique "island architecture" that allows you to use your favorite UI components from React, Vue, Svelte, and more, while only shipping the necessary JavaScript to the browser.

## Why Astro?

1. **Zero JS by default**: Astro ships zero JavaScript to the browser by default.
2. **Component Independent**: Use any UI framework you want.
3. **Edge Ready**: Deploy to anywhere with an adapter.
